# Jonas Sajonas Portfolio

Welcome to the portfolio website of Jonas Sajonas! This website showcases a variety of web development projects and personal achievements.

## Description

This portfolio is built using the **Next.js** framework, a powerful React-based framework for building modern web applications. It leverages **JavaScript** for the frontend and uses **Next.js Server** for server-side rendering and API routes.

## Installation

To get started with the project locally, follow these steps:

1. **Clone the repository**:

   ```bash
   git clone https://github.com/your-username/jonas-sajonas-portfolio.git
   ```
2. **Install dependencies**:
 
  Navigate into the project folder and install all necessary dependencies:

   ```bash
   npm install
   ```
3. **Run the development server**:

  Afther the installation, you can start the Next.js development server:

   ```bash
   npm run dev
   ```
  The site will be available at http://localhost:3000.
